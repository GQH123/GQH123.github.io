BZOJ 3304: [Shoi2005]带限制的最长公共子序列

  • Time Limit: 10 Sec

  • Memory Limit: 128 MB

Description

Input

输入共三行，每行为长度不超过500的，小写字母组成的非空字符串
按顺序分别表示x,y,z

Output

如存在满足条件的N，输出W的长度，否则输出 NO SOLUTION

Sample Input

helloworld

hellxebore

xr

Sample Output

5

HINT

w=hxeor

本题要求找出的W首先是X与Y的公共子序列并且包含Z，然后才是满足这些条件的

字符串里面找最长的。

题目分析
